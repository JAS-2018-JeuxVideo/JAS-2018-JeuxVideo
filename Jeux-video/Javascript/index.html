Javascript

C'est un langage de programmation de scripts orienté objet.

Quezako ?

  • langage de programmation : formalisme permettant d'écrire du code source qui sera par la suite transformé en un programme pouvant être exécuté.
  • programmation de scripts : le code source sera, plus ou moins directement, exécuté par l'ordinateur. Dans certains cas (pas en javascript), le code devra être transformé (ie compilé) pour pouvoir être exécuté par l'ordinateur.
  • Orienté objet : dans le langage, les données pourront être manipulées sous forme d'objets. Un objet sera une façon de décrire des choses en leur associant des caractéristiques ainsi que des manipulateurs (actions modifiant l'état de l'objet, ...).
En pratique, Javascript est un langage qui permettra d'ajouter de l'interactivité à une page web (lui permettre de réagir aux actions de la personne qui la regarde).
