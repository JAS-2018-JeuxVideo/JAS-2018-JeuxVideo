Les variables

Les variables sont un élément prépondérant de la programmation. On peut les voir comme des boites permettant de stocker une donnée qui est ensuite facilement manipulable : on mémorise du contenu (du texte, des chiffres...) en lui attribuant un nom qui permettra d'y accéder au long de notre programme.

Déclarer une variable

L'une des première étape est de déclarer la variable. Cela consiste globalement à lui attribuer un nom pour pouvoir l'utiliser et la repérer par la suite.
Modifiez le fichier test1.js de la façon suivante : 
var b;
document.write ( a );
document.write ( b );
Actualisez (rechargez) le fichier dans le navigateur web.
Quelques explications :
  • Ligne 1, on déclare la variable en utilisant le mot clé var et en donnant le nom de la variable.
  • Ligne 2, on affiche le contenu de la variable a (qui n'existe pas).
  • Ligne 3, on affiche le contenu de la variable b (qui existe mais ne contient rien).

Nommer une variable

Concernant le choix du nom des variables, il est libre, mais ne peut contenir que :
  • des lettres (majuscules ou minuscules) et en général non accentuées (bannir é,è,à,ç...). Dans le cas de d'utilisation de majuscules et minuscules, il faudra respecter la casse;
  • des chiffres (sauf en début de nom);
  • des traits de soulignement _;
  • le signe dollar $;
En général, on fera débuter le nom de variable par un caractère. Si le nom de variable est composé de plusieurs mots, il est courant de mettre une majuscule à chaque mot de manière à les rendre plus visibles (cette notation est appelée camelCase).
Enfin, un certain nombre de mot sont réservés au langage de manière à éviter les erreurs d'interprétation. Des mots comme break, else, delete, this, while, voire même var (voir la liste des mots réservés en annexe).

Affecter une variable

L'affectation d'une variable correspond à mettre une valeur dans la boite. Elle se fait simplement par l'utilisation du signe égal. Modifiez le fichier test1.js de la façon suivante : 
var b = 7;
document.write ( b );
b = b + 7;
document.write ( b );
Actualisez (rechargez) le fichier dans le navigateur web.

Types de variables

Les variables ont un type, c'est-à-dire qu'elles acceptent certains contenus et pas d'autres. Il existe différents types de valeur :
  • entiers : 1, 4, -5, 0, ...
  • réels : 1.345 -2.5, 4.0, ...
  • booléens : true, false
  • textes : "coucou", 'machin', 'r', ...
Ou encore tableaux, objets, fonctions, ... Le typage d'une variable se fait au moment de l'affectation. Dans les exemples précédents : 
var b = 7;
implique que b accepte des nombres entiers; 
var c = "bonjour";
implique que c accepte du texte (des chaînes, string en anglais). C'est la présence de guillemets simples ou doubles qui déterminent cela. Ainsi : 
var d = "7";
considère que 7 est un caractère, non un chiffre ou que 
var e = b;
prendra la valeur qui est celle attribuée à b, type inclus, soit un entier.

Pour connaître le type d'une variable, il est possible d'utiliser typeof. Modifiez le fichier test1.js de la façon suivante : 
var b = 7;
document.write ( typeof(b) );
b = "coucou";
document.write ( typeof(b) );
Actualisez (rechargez) le fichier dans le navigateur web.
