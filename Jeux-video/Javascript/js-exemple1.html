Un exemple simple

Une première page web

En utilisant l'éditeur de texte Bracket (ou tout autre éditeur de texte pur, pas Word ou similaire), faire un nouveau fichier dans lequel vous écrirez : 
<!doctype html>
<html>
    <head>
        <title>Premieres lignes javascript</title>
    </head>
    <body>
        <h1>Débuter avec Javascript</h1>
    </body>
</html>
Sauvegardez (enregistrez) ce fichier sur le bureau en lui donnant le nom test1.html
Ouvrir le fichier avec le navigateur web (double-cliquez sur le fichier sur le bureau).
Quelques explications :
  • Ligne 1, on spécifie le formalisme utilisé. Ici du html dans sa 5ème version.
  • Lignes 2 et 9, toute la spécification de la page se retrouve entre ces deux balises
  • Lignes 3 et 5, encapsulation de l'en-tête de la page
  • Ligne 4, le titre de la page (affiché dans l'onglet du navigateur)
  • Lignes 6 et 8, encapsulation du corps de la page : ce qui s'affichera dans la fenêtre du navigateur
  • Ligne 7, un titre de niveau 1

Avec un mini-script

Modifiez le fichier html de la façon suivante : 
<!doctype html>
<html>
    <head>
        <title>Premieres lignes javascript</title>
    </head>
    <body>
        <h1>Débuter avec Javascript</h1>
        <script>
            document.write("j'affiche un texte dans la page");
        </script>
    </body>
</html>
Actualisez (rechargez) le fichier dans le navigateur web.
Quelques explications :
  • Lignes 8 et 10, encapsulation des scripts de la page
  • Ligne 9, le code source du script :
    • document est un objet qui représente la page web
    • write est une action permettant d'écrire du texte dans la page
    • "j'affiche un texte dans la page" représente les paramètres de l'action (le texte à écrire)
  • Le fait d'écrire document.write explicite le fait de dire : je demande à utiliser l'action write de l'objet document.

Avec un mini-script externe

Créez un nouveau fichier texte contenant : 
document.write("j'affiche un texte dans la page");
Sauvegardez (enregistrez) ce fichier sur le bureau en lui donnant le nom test1.js
Modifiez le fichier html de la façon suivante : 
<!doctype html>
<html>
    <head>
        <title>Premieres lignes javascript</title>
        <script src="test1.js"></script>
    </head>
    <body>
        <h1>Débuter avec Javascript</h1>
    </body>
</html>
Actualisez (rechargez) le fichier dans le navigateur web.
Quelques explications :
  • Ligne 5, encapsulation des scripts de la page depuis un fichier externe
