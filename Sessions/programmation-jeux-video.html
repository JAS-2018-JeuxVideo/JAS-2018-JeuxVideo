Atelier Jeux, Arts et Sciences

Session : Programmation de jeux vidéo

Objectifs : Apprendre la programmation aux enfants/adolescents en leur faisant créer un jeu vidéo plus ou moins simples. Pour ceux qui le souhaitent, selon l'avancement de leurs projets, de présenter leur réalisation lors des journées du numériques (2, 3 et 4 novembre à l'espace rencontre de Millery).
Nous y aborderons différentes techniques de programmation, quelques notions d'anglais (comme le langage utilisé se base sur des mots clefs en anglais) ainsi que l'utilisation des concepts de mathématiques et de physique qu'ils voient en cours. A la fin de l'atelier, les projets aboutis pourront être mis en ligne.

  • Public : à partir du CM2
  • Séances : le vendredi de 18h à 20h, salle des merveilles à la MEJC de Millery
  • Période : du 28/09/2018 au 18/01/2019

Langages utilisés

HTML : qui permettra de décrire la page web hébergeant le jeu

CSS : description de la mise en forme de la page web

Javascript : le langage de programmation qui fera fonctionner le jeu

Les outils

Phaser : framework javascript facilitant la mise en place du jeu

Github : outil/site permettant de gérer le développement du jeu (et sa mise en ligne)

Piskel : outils en ligne permettant de créer des sprites pour les jeux

Tiled : outils permettant de créer des niveaux de jeu à partir de sprites

Bracket : éditeur de texte qui sera utilisé pour le développement

node.js / npm : outils qui permettront de tester le jeu en local sur l'ordinateur avant sa mise en ligne
